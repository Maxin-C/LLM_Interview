# LLM_Interview
存储面试大模型相关岗位整理得到的资料，希望能够节省即将毕业却没有准备思路的朋友能够得到一些帮助。

## hot100解题心得

主要记录了每道题的关键方法，适合场景：

- 刷过题的在面试前快速过一次

- 作为第一次刷题的时候做参考的提示，因为心得写得很概括也没有给代码，如果能够理解`解法`并且自己实现，我认为应该比直接看leetcode题解复制粘贴要有收获得多

此外，`解法`有很多个人观点，我自己都没刷明白，酌情参考

系统地学习leetcode推荐[代码随想录](https://programmercarl.com/),大佬开源且总结得非常系统。建议初学者跟着都刷一遍，hot100应该能做出来80%左右。
